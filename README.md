<h1># 🚗 TripWise Tools</h1>

TripWise Tools is a simple collection of travel utility calculators built using HTML, CSS, and JavaScript.  

This project helps users calculate important travel-related values like speed, arrival time, fuel cost, and total trip expenses using basic math logic.

---

## 🛠 Tools Included

### 1️⃣ Average Speed Calculator
Calculate your travel speed by dividing total distance by total time.

---

### 2️⃣ ETA (Estimated Time of Arrival) Calculator
Estimate your arrival time based on distance and average speed.

---

### 3️⃣ Fuel Cost Calculator
Calculate total fuel expense based on:
- Distance
- Mileage
- Fuel price

---

### 4️⃣ Itinerary Cost Estimator
Get total trip cost by summing multiple expenses like:
- Hotel
- Food
- Travel
- Miscellaneous
