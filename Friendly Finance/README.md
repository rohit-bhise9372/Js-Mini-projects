<h1>💰 Friendly Finance</h1>

Friendly Finance is a collection of simple and practical financial tools designed to help users manage everyday calculations easily.

This project is built using **HTML, CSS, and Vanilla JavaScript**, focusing on logic building, user input handling, and real-world problem solving.

---

## 🚀 Features

### 🔹 1. Bill Batwara (Bill Splitter)
Split expenses easily among friends.
- Enter total bill amount
- Add number of people
- Get per person share instantly

Perfect for trips, parties, and group dinners.

---

### 🔹 2. Currency Converter
Convert one currency into another.
- Enter amount
- Select currencies
- Instantly view converted value

Helps understand real-time value differences.

---

### 🔹 3. Discount Deducer
Calculate final price after discount.
- Enter original price
- Add discount percentage
- Get final payable amount

Useful for shopping and sales calculations.

---

### 🔹 4. Paise Planner
Simple budget planning tool.
- Enter income
- Add expenses
- Calculate remaining balance

Helps in personal financial planning.
