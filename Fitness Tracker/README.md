<h1>🏋️ Fitness Tracker</h1> 

Fitness Tracker is a collection of health-focused mini web applications designed to help users monitor their fitness, nutrition, hydration, and weight goals.

This project is built using **HTML, CSS, and Vanilla JavaScript** and focuses on real-world health calculations, logical problem-solving, and DOM manipulation.

---

## 🚀 Features

### 🔹 1. BMI Calculator

Calculate your Body Mass Index based on:

- Height  
- Weight  
- Gender  

The application calculates BMI and helps users understand their body category (Underweight, Normal, Overweight, etc.).

This tool demonstrates:
- Mathematical formula implementation  
- Conditional logic  
- Dynamic result display  

---

### 🔹 2. Calorie Calculator

Estimate daily calorie requirements based on:

- Age  
- Gender  
- Activity Level  
  - Sedentary  
  - Lightly Active  
  - Active  

The app calculates calorie needs using user profile information and displays recommended intake.

This tool demonstrates:
- User input handling  
- Conditional calculations  
- Health-based logic implementation  

---

### 🔹 3. Water Watcher (Hydration Calculator)

Calculate recommended daily water intake based on:

- Body Weight  
- Current Season  
  - Winter  
  - Summer  
  - Monsoon  

The app adjusts hydration needs depending on seasonal conditions.

This tool demonstrates:
- Real-world logic application  
- Condition-based output  
- Personalized recommendations  

---

### 🔹 4. Step Goal Planner

Plan your weight goal using:

- Current Weight  
- Target Weight  
- Duration (in months)  

Based on the information provided, the app calculates the number of steps required per month to reach the goal weight and displays it on the UI.

This tool demonstrates:
- Goal-based calculation logic  
- Multi-input data processing  
- Dynamic UI updates
